Compass to Dawn

Compass to Dawn was a senior class project that I helped develop. The game itself is a story driven game about a post-apocalyptic world. I developed a text-based language to encode the game's dialogue/decision tree and implemented the tokenizer, parser, and runtime interpreter. The game was my first introduction to Unity. Prior to this, I had only developed games with Bevy (for Rust), Processing (for Java), and raw Javascript.
