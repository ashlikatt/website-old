MapPlanner is a graph storage format intended to assist with developing games with a large number of areas. (As my games tend to have) A small CLI to interact with the format may be found here.


Minecraft Lore Image Generator

This tool generates a Minecraft command to give an item with an image as its lore. It may be found here.


DiamondFire Template Encoder/Decoder
