My Languages

Here is a collection of languages I have made. Some are for programming in, and some are for other purposes. My esolangs page may be found here. This page only shows highlights.


Foreach

Foreach is an esoteric programming language where the only control flow is for-each loops and function calls. The documentation may be found here. The interpreter may be found here.


Cast

Cast is an esoteric programming language where the only operation is unsafely casting user-defined types between eachother. The language ended up being a strange form of functional programming. The documentation may be found here. The interpreter may be found here.


Tupilled

Tupilled is an esoteric programming language I co-designed, where the only datatype is tuples and lambda expressions. It is inspired by functional programming principles. The documentation may be found here.


MapPlanner

MapPlanner is a graph storage format intended to assist with developing games with a large number of areas. (As my games tend to have) A small CLI to interact with the format may be found here.


Terne

Terne is one of the first languages I designed. It is a functional language that supports lambdas, bits, and arrays of bits as the only datatypes. It features a highly dynamic syntax that (unknown to be at the time) makes the language very hard to compile.


Kindling

Kindling is a work-in-progress linker language for DiamondFire. It is heavily inspired by Common Lisp.


Fire

Fire is a work-in-progress language intended to transpile to Kindling for DiamondFire. It is heavily based on Rust and Kotlin.
